Ferramentas Utilizadas

Voltar

Para começar a praticar HTML é necessário instalar o VSCode no seu computador, dentro desta ferramenta podemos criar pastas html, como também de outras tecnologias.

Dicas para facilitar a sua prática:

  • Instale a Extensão Live server, para que as suas páginas atualizem automaticamente.
  • escreva no começo da página "html:5"(sem aspas), para forma uma estrutura inicial da página
  • Como preferência de navegador é recomendado usar o chrome para testar os seus códigos

Inspetor de Elementos

Voltar

A inspeção de elementos é um fator importante para aprender a criar websites, isso pois nós podemos ver como foi criado o site que você está utilizando. Para utilizar essa ferramenta, você deve apertar o botão F12 do seu teclado no site da sua escolha, depois vai abrir uma aba, onde mostrará tanto o código html como também o CSS.

Estrutura básica do HTML

Voltar

A estrutura básica é feita entre as tags html que é onde todo o nosso código estará funcionando, dentro do html teremos támbem as tags head e body, que são respectivamente a configuração e conteúdo do site.

Falando sobre Tags

Voltar

As tags são os comandos do HTML, que servem para criar os conteúdos de um site, eles podem vim com símbolos de menor que e maior que, e dentro delas estará um comando especificado, além disso elas podem aparecer como só tags de abertura ou também podem ter fechamento

Atributos das tags

Voltar

Os atributos são configurações adicionais das tags, promovendo ações/informações a mais, que as tags sozinhas não podem fazer. OBS: Elas sempre ficam dentro das tags de abertura

Textos

Voltar

Para criar um título, nós utilizamos a tag h1, temos também sub-títulos que vão do h2 até no máximo h6.

outras tags para texto são:

  • '<'p'>'-cria parágrafos
  • '<'strong'>'- deixa as palavras em negrito
  • '<'i'>'- deixa as palavras em Itálico
  • '<'blockquote'>'- toda palavra dentro desta tag torna-se uma citação
  • '<'u'>'-Sublinha textos
  • '<'sup'>'-Eleva números e letras
Para todos os tipos de tag é necessário fazer pesquisas no google!

Listas Ordenadas e Não Ordenadas

Voltar

Listas ordenadas: são organizadas em ordem numerica e são formadas por tags '<'ol'>', ''

Listas Não Ordenadas: elas são listas sem ordem, geralmente usadas para listar alguma coisa e são formadas pelas tags '<'ul'>', '/<'ul'>'

Dentro das tags dos dois tipos de listas só podemos escrever cada item entre as tags '<'li'>', ''

links

Voltar

Para utilizar links nos usamos a tag de ancoragem 'a' e seu atributo href="link de sua escolha" e entre as tags nós escrevemos o nome do link

Outros atributos para a tag de link são:

  • '<'title='>'- Quando você passar o mouse por cima do link vai aparecer um texto da sua escolha
  • '<'target='>'- Lhe dá opções de navegação usando o seu link como mudar a página ou criar uma nova aba quando clica no link
